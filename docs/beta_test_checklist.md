# Beta Test Checklist

Use this for every Phase 3 beta user.

- Did install work?
- Did capture work?
- Was setup confusing?
- Was the CLI easy to use?
- Was diagnosis understandable?
- Was diagnosis correct?
- Did the suggested fix help?
- Did the user trust the diagnosis?
- What confused the user?
- Would they use it again?
- Would they pay for it?
- Was an anonymized trace saved in `real_world_cases/`?

Record the answer in `feedback/users.csv`.
